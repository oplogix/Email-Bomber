# Email Bomber



## Features

* Send with Gmail and Outlook
* Or personalize to your email service


## Instructions

Clone the repository:

```
git clone https://github.com/boredcoderyt/Advance-Email-Bomber
```

Navigate to folder:

```
cd Email-Bomber
```

Install requirements:
```
pip install -r requirements.txt
```

Use CLI version:
```
python3 ebomber.py
```
